# Executing Plans

This skill executes structured implementation plans with defined milestones and review checkpoints.

## Execution Framework

### 1. Plan Review
- Verify plan completeness
- Confirm resource availability
- Identify dependencies

### 2. Phase Execution
- Execute tasks sequentially
- Monitor progress
- Document outcomes

### 3. Checkpoint Reviews
- Review completed work
- Validate against acceptance criteria
- Adjust plan if needed

### 4. Final Delivery
- Complete all tasks
- Perform final verification
- Deliver results

## Execution Phases

### Phase 1: Preparation
- Set up development environment
- Review requirements and specifications
- Confirm access to necessary resources

### Phase 2: Implementation
- Execute development tasks
- Follow coding standards
- Perform unit testing

### Phase 3: Testing
- Conduct integration testing
- Validate functionality
- Address bugs and issues

### Phase 4: Delivery
- Deploy to target environment
- Perform final verification
- Document deliverables

## Checkpoint Questions

1. Are we on track with the timeline?
2. Are there any blocking issues?
3. Do we need to adjust the plan?
4. Are deliverables meeting quality standards?
5. Have all acceptance criteria been met?

## Checklist

- [ ] Plan is reviewed and approved
- [ ] Resources are allocated
- [ ] Environment is set up
- [ ] Tasks are completed as scheduled
- [ ] Checkpoints are reviewed
- [ ] Deliverables are complete

## Output Deliverables

- Completed implementation
- Test results
- Documentation
- Final report
