# Writing Plans

This skill helps break down complex requirements into structured, actionable plans.

## Planning Framework

### 1. Requirements Analysis
- Understand the problem
- Identify key requirements
- Define success criteria

### 2. Task Decomposition
- Break into smaller tasks
- Define dependencies
- Estimate effort

### 3. Sequence Planning
- Order tasks logically
- Identify milestones
- Create timeline

### 4. Resource Allocation
- Assign responsibilities
- Identify needed resources
- Plan dependencies

## Plan Structure

### Executive Summary
- Brief overview of the plan
- Key objectives
- Expected outcomes

### Requirements
- Functional requirements
- Non-functional requirements
- Assumptions and constraints

### Tasks
- Task description
- Dependencies
- Estimated effort
- Owner

### Timeline
- Milestones
- Deadlines
- Progress tracking

### Risks
- Potential risks
- Mitigation strategies
- Contingency plans

## Planning Techniques

### Work Breakdown Structure (WBS)
Hierarchical decomposition of tasks into manageable components.

### Gantt Chart
Visual timeline showing task dependencies and progress.

### Critical Path Method
Identify tasks that determine project completion time.

### RACI Matrix
Define roles and responsibilities for each task.

## Checklist

- [ ] Requirements are clearly defined
- [ ] Tasks are properly decomposed
- [ ] Dependencies are identified
- [ ] Timeline is realistic
- [ ] Risks are addressed
- [ ] Responsibilities are assigned

## Output Deliverables

- Detailed plan document
- Task breakdown
- Timeline
- Risk assessment
- Resource allocation
