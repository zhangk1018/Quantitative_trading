# Internal Communications

This skill helps create professional internal communication materials following company standards.

## Document Types

### 1. Status Reports
- Weekly/daily progress updates
- Project status summaries
- Milestone achievements

### 2. Leadership Updates
- Executive briefings
- Strategic updates
- Key metrics and KPIs

### 3. Announcements
- New initiatives
- Process changes
- Important reminders

### 4. Incident Reports
- Root cause analysis
- Impact assessment
- Resolution steps

### 5. FAQs
- Common questions
- Knowledge base articles
- Troubleshooting guides

## Writing Guidelines

### Tone and Style
- Professional but approachable
- Clear and concise
- Consistent formatting
- Audience-appropriate language

### Structure
- Clear subject line
- Executive summary
- Detailed content
- Action items
- Contact information

## Templates

### Status Report Template
```
Subject: [Project Name] Status Update - [Date]

1. Progress Summary
   - Completed tasks
   - In-progress tasks
   - Upcoming tasks

2. Issues/Risks
   - Current blockers
   - Mitigation strategies

3. Next Steps
   - Planned activities
   - Expected deliverables
```

### Incident Report Template
```
Subject: Incident Report - [Incident Name]

1. Summary
   - What happened
   - When it happened
   - Impact

2. Root Cause
   - Analysis findings
   - Contributing factors

3. Resolution
   - Actions taken
   - Preventive measures

4. Lessons Learned
```

## Checklist

- [ ] Document type is appropriate for the audience
- [ ] Content is clear and concise
- [ ] Formatting is consistent
- [ ] Key information is highlighted
- [ ] Action items are clearly stated
- [ ] Review and approval obtained

## Output Deliverables

- Internal communication documents
- Status reports
- Announcements
- Incident reports
- FAQs
