# Phase 8：策略引擎 - 自定义策略编辑器与回测系统

## 8.0 需求背景

### 用户需求
用户希望能够自编量化策略，例如：
1. **回调缩量**：上升趋势内回踩支撑，量能萎缩
2. **箱体放量突破**：底部长期横盘后放量突破
3. **三角收敛向上突破**：末端收敛后放量突破
4. **平台放量突破**：上涨中继平台突破
5. **量能阶梯**：阶梯温和放量小阳爬升

### 功能目标
- ✅ 可视化策略编辑器（拖拽式/积木式）
- ✅ 策略条件组合（AND/OR/NOT 逻辑）
- ✅ 历史回测引擎（计算胜率、收益、最大回撤）
- ✅ 策略信号自动生成
- ✅ 策略库管理（保存/加载/分享）
- ✅ 策略绩效评估报告

## 8.3 数据库设计

### 8.3.1 策略模板表 `strategy_template`

```sql
CREATE TABLE IF NOT EXISTS strategy_template (
    id SERIAL PRIMARY KEY,
    name VARCHAR(128) NOT NULL,
    description TEXT,
    conditions JSONB NOT NULL,      -- 策略条件定义
    parameters JSONB DEFAULT '{}',  -- 策略参数（回测配置等）
    tags TEXT[],                    -- 标签：突破类、回调类、趋势类等
    performance JSONB,              -- 回测绩效缓存
    created_by VARCHAR(64) DEFAULT 'system',
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    UNIQUE(name)
);
```

**conditions JSON 结构**：
```json
{
  "type": "group",
  "operator": "AND",
  "children": [
    {
      "type": "condition",
      "field": "ma5",
      "operator": ">",
      "value": {"field": "ma20"}
    },
    {
      "type": "condition",
      "field": "change_pct",
      "operator": ">",
      "value": {"number": -20}
    },
    {
      "type": "group",
      "operator": "OR",
      "children": [...]
    }
  ]
}
```

### 8.3.2 回测结果表 `backtest_result`

```sql
CREATE TABLE IF NOT EXISTS backtest_result (
    id SERIAL PRIMARY KEY,
    strategy_id INT REFERENCES strategy_template(id),
    start_date DATE NOT NULL,
    end_date DATE NOT NULL,
    initial_capital NUMERIC(12, 2) DEFAULT 1000000,
    transaction_cost NUMERIC(5, 4) DEFAULT 0.001,
    
    -- 绩效指标
    win_rate NUMERIC(5, 2),              -- 胜率
    annualized_return NUMERIC(8, 4),     -- 年化收益
    max_drawdown NUMERIC(8, 4),          -- 最大回撤
    sharpe_ratio NUMERIC(8, 4),          -- 夏普比率
    profit_factor NUMERIC(8, 4),         -- 盈利因子
    total_trades INT,                    -- 交易次数
    avg_profit NUMERIC(10, 2),           -- 平均盈利
    
    -- 结果数据
    equity_curve JSONB,                  -- 权益曲线数据
    trade_logs JSONB,                    -- 交易记录
    holdings JSONB,                      -- 持仓记录
    
    status VARCHAR(32) DEFAULT 'running', -- running/success/failed
    error_message TEXT,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

### 8.3.3 策略信号表 `strategy_signal`

```sql
CREATE TABLE IF NOT EXISTS strategy_signal (
    id SERIAL PRIMARY KEY,
    strategy_id INT REFERENCES strategy_template(id),
    stock_code VARCHAR(10) NOT NULL,
    trade_date DATE NOT NULL,
    signal_type VARCHAR(32) NOT NULL,    -- buy/sell/hold
    signal_strength NUMERIC(5, 2),       -- 信号强度 0-100
    reason TEXT,                         -- 触发原因
    price NUMERIC(10, 2),                -- 触发价格
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    UNIQUE(strategy_id, stock_code, trade_date)
);
```

## 8.4 API 设计

### 8.4.1 策略管理 API

| 方法 | 路径 | 描述 | 参数 |
|:-----|:-----|:-----|:-----|
| GET | `/api/strategy/` | 获取策略列表 | `page`, `page_size`, `tag` |
| POST | `/api/strategy/` | 创建策略 | `name`, `description`, `conditions`, `parameters`, `tags` |
| GET | `/api/strategy/{id}` | 获取策略详情 | `id` |
| PUT | `/api/strategy/{id}` | 更新策略 | `name`, `description`, `conditions`, `parameters`, `tags` |
| DELETE | `/api/strategy/{id}` | 删除策略 | `id` |
| POST | `/api/strategy/{id}/copy` | 复制策略 | `id`, `new_name` |

### 8.4.2 回测 API

| 方法 | 路径 | 描述 | 参数 |
|:-----|:-----|:-----|:-----|
| POST | `/api/backtest/` | 运行回测 | `strategy_id`, `start_date`, `end_date`, `initial_capital`, `transaction_cost` |
| GET | `/api/backtest/{id}` | 获取回测结果 | `id` |
| GET | `/api/backtest/` | 获取回测历史列表 | `strategy_id`, `page`, `page_size` |
| DELETE | `/api/backtest/{id}` | 删除回测记录 | `id` |
| POST | `/api/backtest/compare` | 策略对比回测 | `strategy_ids[]`, `start_date`, `end_date` |

### 8.4.3 信号 API

| 方法 | 路径 | 描述 | 参数 |
|:-----|:-----|:-----|:-----|
| POST | `/api/signal/generate` | 生成策略信号 | `strategy_id`, `trade_date` |
| GET | `/api/signal/` | 获取信号列表 | `strategy_id`, `stock_code`, `trade_date`, `signal_type` |
| GET | `/api/signal/{stock_code}` | 获取股票信号 | `stock_code`, `strategy_ids[]` |
| DELETE | `/api/signal/{id}` | 删除信号 | `id` |

## 8.7 条件表达式设计

### 条件类型

| 类型 | 描述 | 示例 |
|:-----|:-----|:-----|
| **数值条件** | 字段与数值比较 | `close > 10`, `volume_ratio < 0.6` |
| **字段条件** | 字段与字段比较 | `ma5 > ma20`, `high > break_high_20` |
| **布尔条件** | 布尔字段判断 | `is_st == false`, `break_high_20 == true` |
| **统计条件** | 基于统计计算 | `连续上涨天数 >= 3`, `成交量均值 >= 1.5` |
| **模式条件** | 形态/模式匹配 | `pattern_hammer == true`, `pattern_morning_star == true` |

### 操作符支持

| 操作符 | 支持类型 | 说明 |
|:-------|:--------|:-----|
| `==`, `!=` | 所有 | 等于/不等于 |
| `>`, `>=`, `<`, `<=` | 数值/字段 | 比较 |
| `between` | 数值 | 在范围内 |
| `in` | 枚举 | 在列表中 |
| `contains` | 字符串 | 包含 |

## 8.8 回测指标说明

### 核心绩效指标

| 指标 | 计算公式 | 说明 |
|:-----|:--------|:-----|
| **胜率** | 盈利交易数 / 总交易数 | 衡量策略的准确性 |
| **年化收益** | (期末权益/期初权益)^(252/天数) - 1 | 年化收益率 |
| **最大回撤** | max(1 - 权益/历史最高权益) | 最大亏损幅度 |
| **夏普比率** | (年化收益 - 无风险利率) / 收益标准差 | 风险调整后收益 |
| **盈利因子** | 总盈利 / 总亏损绝对值 | 盈利与亏损的比例 |
| **交易次数** | 买入次数 | 策略活跃度 |
