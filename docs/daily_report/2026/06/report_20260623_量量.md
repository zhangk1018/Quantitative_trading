## 三、遇到的问题及解决

| 问题 | 根因 | 解决方案 |
|------|------|----------|
| 任务 6/7 各执行两遍 | 三代 plist 共存 + 2 个 runner 进程并发 | 全部 unload + 文件锁 |
| 断点续跑误跳过任务 | batch_id 自身 running 记录未排除 | SQL 加 `batch_id != 当前batch_id` 条件 |
| 僵尸记录阻塞执行链 | 旧进程异常退出未更新 status | 日志文件 mtime 检测 + 自动清理 |
| daily_basic 只返回 4461 条 | PyWenCai 覆盖范围小于 Baostock | 改 Tushare 优先，Baostock 兜底 |
| pipeline_health_check 失败 | result.error_count() 方法不存在 | 内联计算 error/warn 数量 |
