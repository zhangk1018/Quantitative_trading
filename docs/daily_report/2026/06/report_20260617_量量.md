### 心跳监控修复要点

1. **问题根因**：`query_db_snapshot()` 函数使用 `subprocess.run(['psql', ...])` 调用 psql 命令，但系统 PATH 不包含 Homebrew 安装的 PostgreSQL 路径 `/usr/local/opt/postgresql@18/bin/`

2. **修复方案**：添加 psql 路径自动检测逻辑
   ```python
   # 优先尝试 Homebrew 安装路径
   psql_path = '/usr/local/opt/postgresql@18/bin/psql'
   if not os.path.exists(psql_path):
       psql_path = '/usr/bin/psql'  # 系统默认路径
   if not os.path.exists(psql_path):
       psql_path = 'psql'  # 最后尝试 PATH 中的 psql
   ```

3. **验证结果**：
   - 修复前：`DB，ERR:[Errno 2] No such file or directory`
   - 修复后：`DB，q:06-15(5189) sn:06-15(5189)` ✅
