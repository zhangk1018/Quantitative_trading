### 2. 未覆盖分支说明

`isLargeBody(bar, ratio)` 第 67 行 `if (range === 0) return false`：
- 一字板（range=0）时 `close === open`，既非阳也非阴
- `isMorningStar` / `isEveningStar` 在调用 `isLargeBody` 前已通过 `isBullish` / `isBearish` 过滤
- 因此 range=0 的 K 线不会进入 `isLargeBody` 调用路径
- 该分支为防御性编程（防止除零），属于良好实践，无需额外测试
