### 2. 选股器"看涨吞没(近10天)"不返回数据 Bug 修复

**根因**: `screener.ts` 的 `ScreenerFilterPayload` 类型中 `filterGroup.conditions` 缺少 `lookbackDays` 字段声明，导致 `buildScreeningParams` 无法正确传递 `pattern_bullish_engulfing=10` 参数

**修复**: `ScreenerFilterPayload.filterGroup.conditions` 类型新增 `lookbackDays?: number`

**验证结果**:
- API 参数 `pattern_bullish_engulfing=10` 正确传递 ✅
- 后端返回 2053 只匹配股票 ✅
- Playwright 端到端自测：选股 → 双击股票 → K线弹窗 Canvas 可见 ✅
