### 2. 策略保存/加载功能
- 新增 `SaveStrategyModal`（保存确认弹窗）、`StrategyListDrawer`（策略列表抽屉）
- 新增 `useSavedStrategies` Hook：支持 localStorage 存储、版本管理、输入清洗
- `ScreenerContext` 新增 `LOAD_STRATEGY` action，加载时合并默认值保证向前兼容
- 导出 `StrategyOperationResult` 类型，策略操作失败时明确反馈错误消息

### 3. 三轮代码审阅修复
- **类型安全**：消除 `as` 断言，新增 `isApiErrorLike` / `getErrorMessage` 类型守卫
- **错误处理**：新增 `loadMoreError` 状态 + 底部重试按钮；`timedOutRef` 区分超时 vs 用户取消
- **命名优化**：`reset` → `clearResults`，`refresh` → `fetchFirstPage`，`loadMore` → `fetchNextPage`
- **类型修复**：`debounceTimerRef` 从 `as unknown as number` 改为 `ReturnType<typeof setTimeout>`
- **useLayoutEffect → useEffect**：ref 更新不涉及 UI，避免同步渲染阻塞

### 4. 虚拟滚动
- 引入 `@tanstack/react-virtual`，采用 spacer row 占位法
- 固定行高 36px，overscan 5 行，距底部 5 行自动触发加载更多
- 通过 `loadMoreTriggeredRef` 防重复触发

### 5. Hook 进一步拆分
- `useSelection`（37 行）— 单选/全选/计数
- `useWatchlistAdd`（100 行）— 添加自选弹窗/分组/确认
- `useStrategyUI`（65 行）— 策略模态框/保存/重命名/删除

### 6. Base64 编码存储
- `LocalStorageStrategyStorage` 的 `save` 使用 `btoa(encodeURIComponent(...))` 编码
- `load` 兼容旧格式（未编码 JSON），自动回退解析
