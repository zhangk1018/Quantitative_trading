### 3. Bug 修复
- **Pyodide Worker 命名空间错误**：`pyodide.globals.get('_result')` 从主命名空间读取，但脚本在独立 globals 中执行 → 始终返回 undefined，导致筛选全部失效
- **OHLCV 数据索引映射错误**：`OHLCV_CLOSE=0` 实际是时间戳，传给 Python 脚本的 close 数组全是时间戳而非价格
- **分页 total 覆盖**：虚拟化表格自动触发 fetchNextPage 用原始 total 覆盖了过滤后的 total
- **Pyodide CDN 超时**：jsdelivr CDN 下载 ~12MB WASM 易超时，改为本地托管（public/pyodide/），初始化从 60s+ 降至 2.8s
- **import 白名单误拦截**：`from numpy import array` 被 import 正则误匹配，改为先提取 from...import 再检查独立 import

### 4. 自编指标 end-to-end 验证
- 通过 Playwright 浏览器自动化完整验证：localStorage 预填充 → 选股器选择指标 → 点击开始选股 → 结果验证
- 多阈值验证（>0/ >30/ >50/ >100）确认筛选逻辑正确生效
- 最终验证：今收盘>昨收盘指标，5197 只中筛选出 1819 只（35%），符合市场预期
