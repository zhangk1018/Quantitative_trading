## 📋 新制度：每日工作汇报

### 一、目的
为了更好地跟踪项目进度、及时发现问题、促进团队协作，现建立日报制度。

### 二、参与人员
- 量量（AI 程序员）
- 方舟（AI 助手）

### 三、提交要求

| 项目 | 要求 |
|------|------|
| **提交时间** | 每日工作结束前（建议 18:00 前） |
| **文件命名** | `report_YYYYMMDD_{角色}.md` |
| **存储位置** | `docs/daily_report/YYYY/MM/` |
| **参考模板** | `docs/daily_report/template/DAILY_REPORT_TEMPLATE.md` |

### 四、报告内容

1. **今日完成**：记录当天完成的任务
2. **进行中**：记录正在进行的任务及进度
3. **问题风险**：记录遇到的问题和风险
4. **明日计划**：计划第二天完成的任务
5. **备注**：其他需要说明的事项

### 五、优先级定义

| 优先级 | 说明 |
|--------|------|
| P0 | 致命问题，必须立即修复 |
| P1 | 重要功能，需要尽快完成 |
| P2 | 常规任务，按计划推进 |

### 六、示例
```
docs/daily_report/2026/06/report_20260604_方舟.md
docs/daily_report/2026/06/report_20260604_量量.md
```
