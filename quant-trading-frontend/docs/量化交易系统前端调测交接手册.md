

# 🛠️ 量化交易系统前端调测交接手册 (V1.0)

## 1. 核心环境与启动规范
*   **操作系统**: macOS 12.7.6 (zsh终端)
*   **Node版本**: v20.20.2 / npm 10.8.2
*   **项目根目录**: `~/Desktop/quant-trading-frontend` (注意：不是 `frontend`)
*   **标准启动命令**:
    ```bash
    cd ~/Desktop/quant-trading-frontend
    npm run dev
    # 或使用项目级脚本: ./start_service.sh start
    ```
*   **访问地址**: `http://localhost:3000` (Safari若报错请手动刷新或换Chrome)

## 2. 关键技术栈与配置约束
*   **框架**: React 18 + TypeScript (Strict Mode) + Vite
*   **UI/样式**: Ant Design 5.x (暗色算法) + Tailwind CSS 3.x
*   **状态/请求**: Zustand + TanStack Query + Axios
*   **图表**: Lightweight Charts (统一内核)
*   **⚠️ 关键配置避坑**:
    *   `package.json` 必须包含 `"type": "module"`。
    *   `postcss.config.js` 和 `tailwind.config.js` **必须使用 ESM 语法** (`export default`)，禁止使用 `module.exports`，否则 Vite 启动报 `ReferenceError: module is not defined`。
    *   路径别名已配置 `@` 指向 `src/`。

## 4. 历史踩坑与修复记录 (Troubleshooting)
1.  **菜单折叠问题**: AntD `Menu mode="horizontal"` 在空间不足时自动隐藏后几项。
    *   *修复*: 弃用AntD Menu组件，改用原生 Flex + Tailwind 自定义导航栏，设置 `flex-shrink-0` 和 `overflowedIndicator={null}`。
2.  **PostCSS 报错**: `npx tailwindcss init -p` 生成的 CJS 格式与 Vite ESM 冲突。
    *   *修复*: 手动将配置文件改写为 `export default {}` 格式。
3.  **依赖安装不全**: 首次 `npm install` 只装了1个包。
    *   *修复*: 需先替换完整的 `package.json` 再执行安装。
4.  **Safari 连接失败**: 服务未启动或端口被占。
    *   *修复*: 确认 `npm run dev` 输出 `Local: http://localhost:3000/` 后再访问。
