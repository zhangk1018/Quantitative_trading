# 自编指标开发规范（V1.1）

## 1. 概述

自编指标系统允许用户使用 Python 脚本自定义选股指标，基于 Pyodide（WebAssembly 版 CPython）在浏览器端运行。

**核心设计原则**：
- **单只股票输入**：`calculate` 函数每次只处理**一只股票**的日序列数据（一维数组）
- **系统负责循环**：系统内部对全市场股票逐一调用 `calculate`，用户不需要写外层循环
- **NumPy 可用**：支持 `import numpy as np`，pandas 可用但不推荐（性能差）

## 2. 函数签名规范

### 2.1 必须定义 `calculate` 函数

```python
def calculate(open_prices, high_prices, low_prices, close_prices, volumes):

    自编指标计算函数

        volumes      : list[float] - 成交量序列

    返回值:
        int / float  : 一个数值（选股时取最后一天的值与阈值比较）
        list[float]  : 与输入等长的数值序列（选股时同样取最后一天的值）
    """
    # 你的计算逻辑
    return 0
```

### 2.2 参数说明

| 参数名 | 类型 | 含义 | 顺序 |
|--------|------|------|------|
| 第 1 个 | `list[float]` | 开盘价序列 | `open_prices` |
| 第 2 个 | `list[float]` | 最高价序列 | `high_prices` |
| 第 3 个 | `list[float]` | 最低价序列 | `low_prices` |
| 第 4 个 | `list[float]` | 收盘价序列 | `close_prices` |
| 第 5 个 | `list[float]` | 成交量序列 | `volumes` |

> **重要**：参数名可以自定义（建议含义明确），但**数量和顺序必须是 5 个**，分别对应开/高/低/收/量。

### 2.3 返回值规范

- **返回单值**：`return 1` 或 `return 0.5` —— 选股时直接用这个值与阈值比较
- **返回数组**：`return result_list` —— 选股时取数组的**最后一个有效值**与阈值比较
- **数据不足**：返回 `0` 或 `None`（系统会跳过 None）

## 3. 最简示例

### 3.1 示例 1：收盘价大于 5 日均线（返回布尔值）

```python
import numpy as np

def calculate(open_prices, high_prices, low_prices, close_prices, volumes):
    c = np.array(close_prices, dtype=float)
    n = len(c)
    if n < 5:
        return 0
    ma5 = np.mean(c[-5:])
    return 1 if c[-1] > ma5 else 0
```

### 3.2 示例 2：RSI 指标（返回序列）

```python
import numpy as np

def calculate(open_prices, high_prices, low_prices, close_prices, volumes):
    c = np.array(close_prices, dtype=float)
    n = len(c)
    if n < 15:
        return 0

    # 计算 RSI (SMA 版本)
    delta = np.diff(c, prepend=np.nan)
    gain = np.where(delta > 0, delta, 0.0)
    loss = np.where(delta < 0, -delta, 0.0)

    period = 14
    avg_gain = np.full_like(c, np.nan)
    avg_loss = np.full_like(c, np.nan)

    # 简单移动平均
    for i in range(period, n):
        avg_gain[i] = np.mean(gain[i-period+1:i+1])
        avg_loss[i] = np.mean(loss[i-period+1:i+1])

    rs = np.where(avg_loss != 0, avg_gain / avg_loss, 100.0)
    rsi = 100.0 - 100.0 / (1.0 + rs)
    rsi = np.where(np.isnan(rsi), 50.0, rsi)

    return float(rsi[-1])
```

## 4. 实战示例：6 重条件买入

```python
import numpy as np

def calculate(open_prices, high_prices, low_prices, close_prices, volumes):
    """
    6重条件买入选股指标
    每只股票返回最后一天是否满足买入条件: 1=满足, 0=不满足
    条件:
      1. EMA5 > EMA48 (趋势向上)
      2. DIF > DEA 且 MACD柱 > 0 (MACD金叉且在零轴上方)
      3. RSI 在 25~70 区间 (健康区间)
      4. 成交量 > 1.05 * 15日均量 (放量)
      5. 收盘价 >= 20日最高价 * 0.98 (接近20日新高)
    """
    c = np.array(close_prices, dtype=float)
    h = np.array(high_prices, dtype=float)
    v = np.array(volumes, dtype=float)
    n = len(c)

    if n < 50:
        return 0

    # ========== EMA 函数 ==========
    def ema(arr, period):
        alpha = 2.0 / (period + 1)
        out = np.full_like(arr, np.nan)
        valid_start = 0
        while valid_start < len(arr) and np.isnan(arr[valid_start]):
            valid_start += 1
        if valid_start >= len(arr):
            return out
        out[valid_start] = arr[valid_start]
        for j in range(valid_start + 1, len(arr)):
            if np.isnan(arr[j]):
                out[j] = out[j-1]
            else:
                out[j] = alpha * arr[j] + (1 - alpha) * out[j-1]
        return out

    # ========== 1. 趋势向上 ==========
    ema5 = ema(c, 5)
    ema48 = ema(c, 48)
    trend_up = ema5 > ema48

    # ========== 2. MACD ==========
    ema12 = ema(c, 12)
    ema26 = ema(c, 26)
    dif = ema12 - ema26
    dea = ema(dif, 9)
    macd_bar = (dif - dea) * 2
    macd_gold = (dif > dea) & (macd_bar > 0)

    # ========== 3. RSI (通达信SMA版) ==========
    def tdx_sma(arr, n, m):
        out = np.full_like(arr, np.nan)
        valid_start = 0
        while valid_start < len(arr) and np.isnan(arr[valid_start]):
            valid_start += 1
        if valid_start >= len(arr):
            return out
        out[valid_start] = arr[valid_start]
        for j in range(valid_start + 1, len(arr)):
            if np.isnan(arr[j]):
                out[j] = out[j-1]
            else:
                out[j] = (m * arr[j] + (n - m) * out[j-1]) / n
        return out

    change = np.diff(c, prepend=np.nan)
    up = np.maximum(change, 0)
    down = np.maximum(-change, 0)
    up_filled = np.where(np.isnan(up), 0, up)
    down_filled = np.where(np.isnan(down), 0, down)

    rsi_period = 14
    sma_up = tdx_sma(up_filled, rsi_period, 1)
    sma_down = tdx_sma(down_filled, rsi_period, 1)

    rsi = np.full_like(c, np.nan)
    with np.errstate(divide='ignore', invalid='ignore'):
        rsi = np.where(sma_down != 0, 100.0 - 100.0 / (1.0 + sma_up / sma_down), 100.0)

    rsi_low = 25
    rsi_upper = 70
    rsi_valid = (rsi > rsi_low) & (rsi < rsi_upper)

    # ========== 4. 放量 ==========
    def rolling_mean(arr, window):
        out = np.full_like(arr, np.nan)
        cumsum = np.nancumsum(np.where(np.isnan(arr), 0, arr))
        count = np.cumsum(~np.isnan(arr))
        for j in range(window - 1, len(arr)):
            if count[j] >= window:
                if j - window >= 0:
                    out[j] = (cumsum[j] - cumsum[j - window]) / window
                else:
                    out[j] = cumsum[j] / count[j]
        return out

    vol_avg15 = rolling_mean(v, 15)
    vol_threshold = 1.05
    vol_up = v > vol_threshold * vol_avg15

    # ========== 5. 接近20日新高 ==========
    def rolling_max(arr, window):
        out = np.full_like(arr, np.nan)
        for j in range(window - 1, len(arr)):
            window_slice = arr[j - window + 1:j + 1]
            valid = window_slice[~np.isnan(window_slice)]
            if len(valid) > 0:
                out[j] = np.max(valid)
        return out

    hhv20 = rolling_max(h, 20)
    price_break = c >= hhv20 * 0.98

    # ========== 综合买入条件 ==========
    buy_cond = trend_up & macd_gold & rsi_valid & vol_up & price_break

    # 取最后一个有效值
    last_val = 0
    for j in range(n - 1, -1, -1):
        if not np.isnan(buy_cond[j]):
            last_val = 1 if buy_cond[j] else 0
            break

    return last_val
```

## 5. 常见错误与陷阱

### ❌ 错误写法：二维数组 + 外层循环（最常见错误）

```python
# 错误！系统传入的是单只股票的一维数组，不是多只股票的二维数组
def calculate(open_prices, high_prices, low_prices, close_prices, volumes):
    result = []
    for i in range(len(close_prices)):  # ❌ 这是按"天"循环，不是按股票
        c = np.array(close_prices[i], dtype=float)  # ❌ close_prices[i] 是单个数字！
        ...
    return result  # ❌ 返回了天数个元素的列表，但每元素的计算都是错的
```

**为什么错**：`close_prices` 是长度为 N 的一维数组（N=交易日天数），`close_prices[i]` 是第 i 天的收盘价（单个数字），不是第 i 只股票的数据。

### ✅ 正确写法：直接使用整个数组

```python
def calculate(open_prices, high_prices, low_prices, close_prices, volumes):
    c = np.array(close_prices, dtype=float)  # ✅ 整个数组 = 这只股票的所有天
    h = np.array(high_prices, dtype=float)
    v = np.array(volumes, dtype=float)
    n = len(c)  # ✅ n = 交易日天数

    if n < 50:
        return 0

    # 你的计算逻辑...
    return 1 if condition else 0
```

### ❌ 错误写法：引用不存在的 `stock_idx`

```python
# 错误！不存在 stock_idx 变量
c = np.array(close[stock_idx])
```

**正确**：直接使用输入参数数组。

### ⚠️ 注意：数据量

系统向 Pyodide 传入的 OHLCV 数据大约是 **最近 200 个交易日**（约 1 年）。如果你的指标需要更长周期（如 250 日均线），请做好数据不足的处理。

## 6. 运算符与阈值

自编指标的筛选条件支持以下运算符：

| 运算符 | 含义 | 阈值类型 | 示例（RSI 指标） |
|--------|------|----------|------------------|
| `>` | 大于 | 单值 | RSI > 30 |
| `>=` | 大于等于 | 单值 | RSI >= 70 |
| `<` | 小于 | 单值 | RSI < 70 |
| `<=` | 小于等于 | 单值 | RSI <= 30 |
| `==` | 等于 | 单值 | 信号 == 1 |
| `range` | 区间内 | 双值 [min, max] | 30 <= RSI <= 70 |
| `cross_up` | 上穿 | 双值 | RSI 上穿 30 |
| `cross_down` | 下穿 | 双值 | RSI 下穿 70 |

> 注意：`cross_up` / `cross_down` 需要返回数组序列才能判断，仅返回单值的指标不支持。

## 7. 性能建议

1. **优先使用 NumPy 向量化运算**，避免 Python 级别的 for 循环
2. **数据不足时尽早返回**，减少无效计算
3. **避免使用 pandas**（加载慢、内存占用大），NumPy 足够
4. **函数内部定义的辅助函数**（如 `ema()`）每次调用都会重新定义，尽量精简

---

## 8. 校验规则

保存指标时系统会自动校验，校验项包括：

### 错误级（不通过无法保存）
- 公式不能为空
- 公式长度 ≤ 8000 字符
- 括号必须配对
- 不得包含危险关键字（`exec` / `eval` / `os.` / `subprocess` 等）
- 只能 import `numpy` 或 `pandas`
- **必须定义 `calculate` 函数**
- **`calculate` 必须有 5 个参数**

### 警告级（提醒但可保存）
- 参数名建议包含 open/high/low/close/volume 关键词
- 检测到疑似二维数组用法（`close[i]`、`for i in range(len(close))` 等）
- 检测到 `stock_idx` 变量（已废弃）
- 检测到 `result.append() + return result` 模式（请确认返回值含义）

## 9. 调试技巧

1. **用 `print()` 调试**：Pyodide 的 print 输出会显示在浏览器 Console（F12）中
2. **本地验证**：可以先用本地 Python + 真实数据验证指标逻辑，再粘贴到前端
3. **简化测试**：先写最简版本（如直接 `return 1`），确认能选中股票后再逐步加条件
4. **Console 查看错误**：如果选股异常，打开浏览器 DevTools → Console 查看 Python 报错
